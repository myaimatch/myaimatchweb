Design Principles
🌑
Dark First
Pure blacks (#000, #0d0d0d) as the base. Never use dark grays as page backgrounds — true black maximises contrast with purple glows.
💜
Purple-Led
#814ac8 for all interactive elements, #df7afe for glows and accents. Never introduce secondary brand colors — purple is the only hue family.
✦
Glow as Depth
Radial purple glows create spatial hierarchy. Beams at top-center of sections. Card glows bloom on hover. Never flat shadows.
○
Pill-Native
All CTAs, navbars, and inputs use border-radius: 999px. Cards use 18px. Only inner UI elements drop to 8–12px.
⟡
Glass Surfaces
Cards: rgba(255,255,255,.04) + 1px rgba border. Navbar: backdrop-blur: 10px. Never opaque surfaces — layers should breathe.
◌
Trust Through Space
Generous whitespace signals confidence. 48–64px section padding. Text at 40–60% opacity for hierarchy. Never crowd the canvas.
