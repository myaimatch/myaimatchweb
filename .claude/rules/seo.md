# SEO Requirements

Requisitos obligatorios para cada página nueva.

## Meta tags mínimos

- `<title>`
- `<meta name="description">`
- Open Graph: `og:title`, `og:description`, `og:url`, `og:type`

## Estructura de URLs

- Tools: `/tools/[slug]`
- Comparisons: `/compare/[tool-a]-vs-[tool-b]`
- Blog: `/blog/[slug]`

## Structured data

- JSON-LD en tool detail pages.
- JSON-LD en comparison pages.

Las rutas completas viven en `.claude/context/routes.md`.
