# Project Overview

## Qué es MyAIMatch

AI Tools Directory + Assessment Funnel en `myaimatch.ai`.

Universal AI matching engine + consultancy que elimina "AI overwhelm". La plataforma une discovery + implementation.

## Funnel

1. **Free Assessment** (Typeform) — captura y perfilado.
2. **Personalized Recommendation** (Claude API vía Zapier) — match de herramientas.
3. **Paid Strategy Roadmap** — consultoría de arquitectura.
4. **AI Stack Setup** — implementación guiada.
