# Stack

| Layer | Tool |
|---|---|
| Framework | Next.js 14 (App Router) |
| Styling | Tailwind CSS + shadcn/ui |
| Data / CMS | Airtable API |
| AI | Claude API (Anthropic) |
| Forms | Typeform |
| Automation | Zapier |
| Email | Klaviyo |
| Payments | Stripe |
| Scheduling | Cal.com |

## Comandos

```bash
npm run dev       # dev server en localhost:3000
npm run build     # verificar build — correr después de cada cambio
npm run lint      # lint
```
