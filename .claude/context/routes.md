# Routes

| Route | Purpose |
|---|---|
| `/` | Homepage: hero, search, category grid, deals, assessment CTA |
| `/directory` | Tool grid con filtros y categorías (data desde Airtable) |
| `/tools/[slug]` | Tool detail pages (static, generadas desde Airtable) |
| `/compare/[tool-a]-vs-[tool-b]` | Side-by-side comparison (Versus-style, SEO URL) |
| `/assessment` | Opener route que lanza el popup del Free AI Match (Typeform) |
| `/services` | Overview de servicios: strategy, implementation y fractional AI leadership |
| `/ai-implementation` | AI Implementation & Systems Setup page con Cal.com |
| `/services/ai-tools-implementation` | Redirect legacy hacia `/ai-implementation` |
| `/blog/[slug]` | MDX blog posts (comparison articles "Tool A vs Tool B") |
